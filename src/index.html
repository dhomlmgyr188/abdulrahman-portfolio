تطوير لوحة تحكم لمتجر إلكتروني Front-End

لوحة تحكم لمتجر إلكتروني تتيح إدارة الموقع بسهولة، حيث يمكن إضافة مستخدمين جدد وإدارة بياناتهم، وكذلك إضافة منتجات جديدة وتحديثها. كما توفر عرضًا لإحصائيات الموقع لمتابعة الأداء، بالإضافة إلى إدارة الطلبات ومتابعة حالتها بشكل منظم.

تطوير متجر الكتروني Front-End

متجر إلكتروني لبيع الملابس يوفّر تجربة تسوق سهلة ومنظمة للمستخدمين، حيث يمكن تصفح المنتجات والبحث عنها باستخدام خاصية البحث المتقدم. يتيح المتجر إضافة المنتجات إلى عربة الشراء وإدارتها قبل إتمام الطلب، كما يحتوي على صفحة دفع لإكمال عملية الشراء بشكل سلس وآمن.

التقنيات المستخدمة:

npm webpack html sass-scss js bootstrap
Github Live Demo

تطوير موقع مدرسة علمني الإبتدائية Front-End

موقع مدرسي تم تطويره لعرض معلومات المدرسة بشكل منظم، حيث يتيح التعرف على بيانات المدرسة وطلابها، وعرض الطلاب المتميزين. كما يوفر معلومات عن موقع المدرسة ووسائل التواصل، مما يسهل الوصول والتواصل مع إدارة المدرسة.

التقنيات المستخدمة:

npm webpack html sass-scss js jQuery bootstrap
